# 梦核：第六层 (Dreamcore: Level 6)

> 你以为你在玩游戏，其实游戏在玩你。你以为你通关了，其实你才刚刚开始。

这是一次被包装成 HTML 小游戏的心理实验，深度致敬电影《盗梦空间》（Inception）。游戏通过六层嵌套的梦境结构，不断反转玩家的认知，挑战你对“现实”的定义。

---

## 🌀 核心机制：梦中梦嵌套

游戏共有 6 层（致敬 Limbo）。每一层都是一个独立的“小游戏”，但每当你以为通关时，都会有新的证据告诉你——你只是醒到了更高一层的梦里。

---

## 🕹️ 层级指南

### 第 0 层：欺骗层 (Deception)
*   **伪装**：一个简单的“颜色反应测试”。
*   **真相**：达到一定分数后，系统将检测到你的“脑电波异常”，揭示这只是潜意识投射。

### 第 1 层：记忆碎片层 (Memory)
*   **玩法**：拼凑一张破碎的脸。
*   **反转**：拼完后你会发现，那张脸正是你自己的投影。

### 第 2 层：选择悖论层 (Paradox)
*   **玩法**：面对一系列极端的道德选择题（电车难题）。
*   **反转**：系统会告诉你，你的所有选择都符合预设模型，你并没有所谓的“自由意志”。

### 第 3 层：折叠走廊 (Folded Corridor)
*   **玩法**：第一人称 3D 走廊探索。
*   **陷阱**：
    *   **空间折叠**：转身后前进后退键会发生翻转。
    *   **诡异计步器**：弹出的步数数字不可信，甚至会出现 0。
    *   **裂纹重复**：暗示你一直在原地打转。
*   **真正出口**：放弃寻找，**站在原地不动 10 秒**。

### 第 4 层：AI 觉醒层 (Awakening)
*   **玩法**：与名为“柯布”的 AI 助手对话。
*   **最大反转**：对话过程中，对方会揭露自己并非 AI，而是被困在这一层的“上一个玩家”。

### 第 5 层：现实回归 (Reality?)
*   **玩法**：尝试关闭游戏并返回现实。
*   **最终选择**：
    *   **接受现实**：退出游戏，回到一个可能是现实的页面。
    *   **我不信**：重新回到第 0 层，开启新的循环。注意：循环中的 NPC 会记住你。

---

## 🎨 视觉与氛围
*   **CRT 特效**：模拟陈旧显示器的扫描线与暗角，营造压抑的科技感。
*   **Glitch 效果**：层级切换时的画面撕裂感，暗示梦境的不稳定性。
*   **3D 沉浸**：第 3 层采用了原生 CSS 3D 渲染，无需外部插件。

---

## 🛠️ 技术实现
*   **Frontend**: HTML5, CSS3 (3D Transforms, Animations), Vanilla JavaScript.
*   **Storage**: localStorage 用于记录梦境循环状态。
*   **Design**: 纯手绘 Canvas 与 3D 布局。
